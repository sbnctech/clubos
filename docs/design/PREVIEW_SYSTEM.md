# Preview System Design

Real-time preview throughout the page and email layout systems.

---

## Preview Principle

> **Users should never publish something they haven't seen.**

Every editing context provides preview that matches the final output.

---

## Preview Contexts

| Context | Preview Shows | Key Considerations |
|---------|---------------|-------------------|
| **Page Editor** | Page as visitors will see it | Theme, layout, responsive |
| **Email Editor** | Email as it appears in inbox | Email client rendering |
| **Layout Settings** | Site with new layout applied | Navigation, auth position |
| **Theme Settings** | Site with new colors/fonts | All pages affected |
| **Block Editing** | Block in page context | Surrounding content |

---

4. **Image blocking** - Some clients block images by default

### Preview Iframe

```tsx
// PreviewFrame.tsx

function PreviewFrame({ pageId, draftContent, theme, layout }) {
  const iframeRef = useRef<HTMLIFrameElement>(null);

  // Send updates to iframe
  useEffect(() => {
    iframeRef.current?.contentWindow?.postMessage(
      {
        type: "PREVIEW_UPDATE",
        payload: { content: draftContent, theme, layout },
      },
      window.location.origin
    );
  }, [draftContent, theme, layout]);

  return (
    <iframe
      ref={iframeRef}
      src={`/preview/${pageId}?mode=draft`}
      className="preview-frame"
    />
  );
}
```

### Preview Route

```tsx
// src/app/preview/[pageId]/page.tsx

export default function PreviewPage({ params }) {
  const [content, setContent] = useState(null);

  // Listen for editor updates
  useEffect(() => {
    const handler = (event: MessageEvent) => {
      if (event.data.type === "PREVIEW_UPDATE") {
        setContent(event.data.payload);
      }
    };
    window.addEventListener("message", handler);
    return () => window.removeEventListener("message", handler);
  }, []);

  // Render page with draft content
  return (
    <PageRenderer
      content={content?.content}
      theme={content?.theme}
      layout={content?.layout}
      isPreview={true}
    />
  );
}
```

## Summary: Preview Everywhere

| Context | Preview Type | Key Feature |
|---------|--------------|-------------|
| **Page Editor** | Split view, full preview | Responsive breakpoints |
| **Email Editor** | Email client simulation | "Images off" mode, test send |
| **Layout Settings** | Before/after comparison | Live site preview |
| **Theme Settings** | Live color/font preview | Multiple pages |
| **Block Editing** | Block in page context | Highlighted in context |
| **Before Publish** | Full preview required | "Preview before publish" |

**Principle:** Nothing gets published without being previewed first.
