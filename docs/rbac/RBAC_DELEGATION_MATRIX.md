# RBAC Delegation Matrix

| Role | Can Delegate | Scope | Notes |
|-----|-------------|-------|------|
| President | All | Global | Audit required |
| VP Activities | Event Chairs | Activities only | Time-bounded |
| Event Chair | None | N/A | Cannot delegate |
| Finance Manager | None | N/A | Cannot grant roles |
| VP Finance | None | N/A | Approval only |
| Board | President | Governance | Recorded vote |

## Hard Denials
- Chairs cannot assign roles
- No cross-domain delegation
- Finance approvals cannot grant permissions
